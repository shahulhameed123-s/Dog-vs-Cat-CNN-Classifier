# Dog vs Cat Image Classifier using CNN

## Project Overview

This project uses a Convolutional Neural Network (CNN) built with TensorFlow and Keras to classify images as either Cats or Dogs.

## Features

- Image classification using CNN
- Data preprocessing with ImageDataGenerator
- Data augmentation
- Model training and validation
- Model saving
- Prediction on new images

## Dataset

Microsoft Cats vs Dogs Dataset

## Technologies Used

- Python
- TensorFlow
- Keras
- Google Colab
- NumPy
- Matplotlib

## Accuracy

Validation Accuracy: Approximately 76%

## How to Run

1. Install dependencies

```
pip install -r requirements.txt
```

2. Open the notebook

```
dog_vs_cat_classifier.ipynb
```

3. Run all cells.
